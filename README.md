# WordPress Website Content Management Project

## Project Overview

This project involved the structured development and management of a WordPress-based website for a humanitarian organisation, Action Against Hunger. The objective was to design a functional, user-friendly platform that supports donations, stakeholder engagement, and public communication.

The project applied core Content Management System (CMS) principles including information hierarchy, intuitive navigation, data validation, usability, and accessibility across devices.

---

## Project Objectives

- Develop a structured website using WordPress
- Implement logical navigation architecture
- Enable donation functionality
- Configure newsletter subscription system
- Implement contact and inquiry management
- Integrate Google Maps for geographic accessibility
- Apply authentication and user dashboard functionality
- Ensure mobile responsiveness
- Conduct structured system testing

---

## Key Functionalities Implemented

- Donation workflow with validation and confirmation
- Newsletter subscription with database capture
- Contact form with backend entry validation
- Donor dashboard for profile updates and donation history
- FAQ interactive toggle section
- Google Maps branch location integration
- Admin authentication system
- Mobile responsiveness testing
- Web browser compatibility testing

---

## Testing & Validation

A structured test plan was conducted to validate:

- Correct vs incorrect data input handling
- Access control authentication
- Form submission validation
- Mobile responsiveness
- Cross-browser compatibility
- Font readability and UI clarity

---

## Design Considerations

- Recognition over recall for usability
- User-centered content structuring
- Clear navigation hierarchy
- Accessible typography and layout
- Lightweight design for better performance

---

## Challenges Encountered

- Plugin compatibility limitations due to PHP version
- Newsletter popup configuration conflicts
- Data capture limitations on certain donation integrations
- Theme readability adjustments

---

## Tools & Technologies Used

- WordPress CMS
- XAMPP (Local server environment)
- PHP (Server configuration context)
- WordPress Plugins
- Database management
- Basic HTML formatting
- Google Maps integration
